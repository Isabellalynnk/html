HTML History:

Tim Berner-Lee was the creator of HTML in the 1990's. He created this language to make it easier to create electronic documents that can be displayed on the web. The name "Hypertext Markup Language" came upon by incorporating the ability to link documents together by electronically using hyperlinks but making it easier as a shortcut.





CSS History:

In 1994 Hakon Wium Lie and Tim Berner-Lee were working together trying to figure out the CSS configuration. They offically released it as a world wide web December 17, 1996. CSS was created by Hakon WIum Lie and Bert Boss; developed by Argo Web Browser.









JavaScript History:

Brendan Eich is the creator of the programming lanuage called JavaScript. Thie language was purposly created for the Netscape Navigator in the 1995. As this language grew it became known as the "LiveScript" later to be change to recognize Netscape's support of having Java within their browser.
